CosuLib for Lua UIs

CosuLib is a Lua module that allows to design user interfaces dynamically. Because of its design, it can be adjusted and extended to many needs as well as rendered in different ways and environments, being a TUI for ComputerCraft, xterm or GUIs via mappings.
