Functions

Component.setBG (c)
Sets the background of this component. In addition to the numbers of the colors API, {renderer.translucent} can be used too for making that layer translucent. If the default color should be used, nil can be used.

Parameters:

  • c number|string|nil The color, {renderer.translucent} for translucent or nil for default.
Component.setFG (c)
Sets the foreground of this component. In addition to the numbers of the colors API, {renderer.translucent} can be used too for making that layer translucent. If the default color should be used, nil can be used.

Parameters:

  • c number|string|nil The color, {renderer.translucent} for translucent or nil for default.
