Module cosu.base

Returns basic functions for cosu.

This module will be called on initial state of cosu (@see cosu). It contains the basic structure of the actual module.

cosu.util.search (t, value, getCondition, bSimilar)
Searches for a value in a table via binary search. Returns the index of the value or nil if not found.

Parameters:

  • t table The table that will be searched for the value.
  • value any The wanted value.
  • getCondition ? function If the elements inside the given table has some specific way to compare for, this function can be provided in addition.
  • bSimilar ? boolean Whenever it should take the next best value it can find or return nil. False by default.

Returns:

    number|nil index The index of the value in the table or nil if not found.
cosu.util.genSetFunctions (Class)
Converts functions that don't return anything in a given Class to return their own Class. This way, the Classes methods can be mostly chained.

Parameters:

  • Class table The Class with the methods to convert.
