Module cosu.component.Grid

Stores other Component's (childs) and handles passing over events and such.

Creator

cosu.new.Grid (nX, nY, nW, nH) Creates a new Grid.

Functions

add (Component) Adds a Component to the Grid.
remove (Component) Removes a Component from the Grid.

unsubscribe (Component, sEvent) Unsubscribes an Component for an type of event.

Creator

cosu.new.Grid (nX, nY, nW, nH)
Creates a new Grid.

Parameters:

  • nX number The x position of the upper left corner.
  • nY number The y position of the upper left corner.
  • nW number The width of the component.
  • nH number The height of the component.

Returns:

    table Grid A new Grid class.

Functions

add (Component)
Adds a Component to the Grid.

Parameters:

  • Component table The Component to add. Will remove it from any other Grid it got added to before.
remove (Component)
Removes a Component from the Grid.

Parameters:

  • Component table The Component to remove.
getContent ()
Returns the Component childs the Grid contains.

Returns:

    table childs List of all childs the Grid has.
subscribe (Component, sEvent)
Subscribes an Component for an type of event this Grid may receive. Those may be passed to the Component, if the Grid does not already listen for them by itself.

Parameters:

  • Component table The Component to subscribe.
  • sEvent string The event that the Component wants to receive.
unsubscribe (Component, sEvent)
Unsubscribes an Component for an type of event.

Parameters:

  • Component table The Component to subscribe.
  • sEvent string The type of event the Component should be unsubscribed to.
pass (...)
Passes an event to the Grid's childs. The childs may handle further actions. If the Grid listens for the same event however, it wont pass it any further.

Parameters:

  • ... any Arguments of the event, including the name.
